personalized machine learning

There is a need, now more than ever, for personalized models and applications that can efficiently leverage big data and adapt to each individual, especially in smart healthcare and medicine applications. PML is a novel shift from the traditional ML paradigm - instead of using 'one-size-fits-all' machine learning models, the PML algorithms are designed so that they provide optimal performance (e.g., accuracy of mood prediction) for each individual in target (risk) group - and not only the average group outcome. Central to this is how to effectively and intelligently extract the person information and embed it into machine learning algorithms to build personalized models that provide optimal performance for each individual. To teach this new modeling paradigm, in 2017, Dr Oggi Rudovic and Prof Roz Picard created a graduate course at MIT Media Lab - MAS.S61: Personalized Machine Learning (PML).

For more details, see: courses.media.mit.edu/2017spring/mass61.
